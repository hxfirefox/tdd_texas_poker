德州扑克
========

#背景:

德州扑克采用52张扑克，分为四个花色（将红桃，方块，黑桃，梅花简化为A，B，C，D），每个花色13张牌，数字从2到14（将J，Q，K，A简化为11，12，13，14）。游戏过程玩家随机抽取5张，电脑随机抽取5张，比较大小。

#需求:

1. 按以下规则对比玩家和电脑的牌大小： 

 I. 规则一：比较各自五张牌中数字最大的（散牌） 
 
 - Example 1 ：“B11, A2, A9, D10, A8” < “D4, A8, C5, C6, B12” 
 - Example 2 ：“D3, D4, D5, D6, D7” < “D5, A6, C7, C8, B9” 
 
 II. 规则二：若散牌中数字最大的牌相等，则比较第二数字大牌
 
 - Example 1 ：“B11, A2, A9, D10, A8” > “D4, A8, C10, C6, B11” 
 
 II. 规则二：若有两张相同数字牌（对子），对子大于散牌，对子之间比较其数字大小；若对子数字相同，将其他牌按散牌比较；  

 - Example 1 ：“B11, A9, D9, D5, A8” < “D4, B10, C10, C6, B12” 
 - Example 2 ：“B11, A9, D9, D5, A8 ” < “D10, A6, C9, B12, B9”

 III. 规则三：若五张牌花色相同（同花），同花大于对子，若都是同花，则依次按对子和散牌比较；
 
 - Example 1 ：“B11, A2, A9, D10, A8” < “D4, A8, C5, C6, B12” 
 - Example 2 ：“D3, D4, D5, D6, D7” < “B5, A6, C7, C8, B9”
 
2. 需要输出两副牌的牌面（如A5，D11 ），及比较的结果
